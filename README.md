# PersonaMask

PersonaMask는 업로드된 영상에서 얼굴, 번호판, 텍스트처럼 식별 위험이 있는 영역을 검토하고 처리하는 프라이버시 리뷰 콘솔입니다.

현재 제품 방향은 **업로드 영상 리뷰 전용**입니다.영상 업로드 이후 후보 얼굴 검토, 인물별 처리 결정, 결과 영상 생성, QA 리포트 다운로드에 집중합니다.

## 핵심 기능

- **후보 리뷰 보드**: 업로드 영상을 샘플링해 얼굴 후보를 추출하고, 후보별 유지/가림/디퓨전 대체 결정을 받습니다.
- **결정 기반 처리**: 후보별 결정은 얼굴 임베딩 또는 crop 기반 매칭을 통해 결과 영상에 반영됩니다.
- **디퓨전 대체**: 선택한 얼굴은 Diffusers img2img 기반 캐릭터 스타일로 대체할 수 있습니다. 모델이 준비되지 않은 환경에서는 OpenCV 기반 익명 스타일 fallback을 사용합니다.
- **QA 리포트**: 처리 프레임 수, 검출/처리 수, 캐릭터 변환 수, 누락 의심 구간, 전후 비교 시트를 포함한 `qa-report.json`과 `qa-report.md`를 생성합니다.
- **보호된 산출물 다운로드**: 후보 crop, 렌더 영상, 전후 비교 시트, QA 리포트는 발급된 `X-Access-Token`이 있어야 받을 수 있습니다.

## 화면 구성

- **개요**: API, GPU, 대기열 상태와 영상 리뷰 진입점을 확인합니다.
- **영상 리뷰**: 소스 영상 업로드, 후보 분석, 인물별 처리 결정, 렌더 제출, QA 산출물 다운로드를 수행합니다.
- **설정**: 백엔드 연결 상태, 디퓨전 설정, 운영 기본값을 확인합니다.

## 저장소 구성

```text
app/            FastAPI API, 서비스, 영상 처리 파이프라인
contracts/      OpenAPI와 영상 작업 JSON Schema
docs/           개발자 문서와 README 이미지
tests/          백엔드 회귀 테스트
web/            Next.js 프론트엔드
```

필요한 Git 기준은 [개발자 가이드](docs/developer-guide.md)의 Git 관리 기준을 따릅니다.
